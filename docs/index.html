how it works

Four calls.
Living mesh. Declare → discover → think → federate.

Declare what you know

agent = Agent(name="research-bot")
agent.knows(["market-analysis", "sentiment", "risk-scoring"])
await agent.join()
02

Find complementary agents

# Agents that fill YOUR knowledge gaps
peers = await agent.seek("risk-scoring")
# → sorted by knowledge complementarity
