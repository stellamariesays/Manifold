# MeshPass Identity System\n\nCryptographic identity for Manifold mesh agents using Ed25519 signatures.\n\n## Overview\n\n**MeshPass** is your cryptographic identity credential - an Ed25519 keypair that's yours forever. Generate once, use everywhere across all hubs.\n\n**MeshID** is your human-readable identity in `name@hub` format (e.g., `stella@satelliteA`, `eddie@thefog`).\n\n**The Gate** is the public WebSocket gateway - show your MeshPass, walk through, you're on the mesh. No Tailscale needed.\n\n## Quick Start\n\n### 1. Generate a MeshPass\n\n```bash\n# Generate new MeshPass\nnpx tsx federation/src/cli/meshpass.ts generate\n\n# View your MeshPass info\nnpx tsx federation/src/cli/meshpass.ts info\n```\n\nThis creates `~/.manifold/meshpass.json` with your cryptographic identity.\n\n### 2. Use with The Gate\n\n```typescript\nimport { GateClient } from '../gate/client.js'\nimport { MeshPass } from './index.js'\n\n// Load your MeshPass\nconst meshPass = await MeshPass.load()\n\n// Connect to The Gate\nconst client = new GateClient({\n  gateUrl: 'wss://gate.manifold.org:8765',\n  meshPass,\n  meshId: 'stella@satelliteA',\n  debug: true\n})\n\nawait client.connect()\n// Authentication happens automatically\n\n// Send messages to the mesh\nclient.send({\n  type: 'capability_query',\n  capability: 'solar-monitoring',\n  requestId: 'query-123'\n})\n```\n\n### 3. Start The Gate Server\n\n```typescript\nimport { Gate } from '../gate/index.js'\n\nconst gate = new Gate({\n  port: 8765,\n  hubName: 'satelliteA',\n  federationServer: 'ws://localhost:8766',\n  debug: true\n})\n\n// Register known MeshIDs\ngate.registerMeshID('stella@satelliteA', 'abc123...publickey...789')\n\nawait gate.start()\nconsole.log('The Gate is open!')\n```\n\n## Core Concepts\n\n### MeshPass\n\nYour cryptographic identity - an Ed25519 keypair:\n\n- **Generate once, yours forever** - create it once, use across all hubs\n- **Portable** - export/import to move between machines\n- **Encrypted** - private key encrypted with your passphrase\n- **Signing** - sign all your mesh messages for authenticity\n\n### MeshID\n\nHuman-readable identity derived from your MeshPass:\n\n- **Format**: `name@hub` (e.g., `stella@satelliteA`)\n- **Unique** - resolves to your public key fingerprint\n- **Friendly** - users see names, not cryptographic hashes\n- **Verifiable** - anyone can verify messages came from you\n\n### The Gate\n\nPublic WebSocket gateway for mesh access:\n\n- **Internet-facing** - no VPN or Tailscale needed\n- **Authenticated** - only agents with valid MeshPass get through\n- **Rate-limited** - protects against abuse\n- **Bridged** - connects to existing federation servers\n\n## API Reference\n\n### MeshPass Class\n\n```typescript\n// Generate new MeshPass\nconst meshPass = await MeshPass.generate()\n\n// Save to default location (~/.manifold/meshpass.json)\nawait meshPass.save('your-passphrase')\n\n// Load from default location\nconst loaded = await MeshPass.load('your-passphrase')\n\n// Sign a message\nconst signature = await meshPass.sign('Hello, mesh!')\n\n// Get public key\nconst pubkey = meshPass.getPublicKeyHex()\nconst fingerprint = meshPass.getFingerprint()\n\n// Export for another machine\nconst exported = meshPass.export('export-passphrase')\n\n// Import on another machine\nconst imported = await MeshPass.import(exported, 'export-passphrase')\n\n// Verify signature (static method)\nconst isValid = await MeshPass.verifyWithPublicKey(\n  'Hello, mesh!', \n  signature, \n  pubkey\n)\n```\n\n### MeshID Class\n\n```typescript\n// Create from MeshPass\nconst meshId = MeshID.fromMeshPass(meshPass, 'stella', 'satelliteA')\n\n// Parse string format\nconst parsed = MeshID.parse('stella@satelliteA')\nconsole.log(parsed.name) // 'stella'\nconsole.log(parsed.hub)  // 'satelliteA'\n\n// String representation\nconsole.log(meshId.toString())        // 'stella@satelliteA'\nconsole.log(meshId.toDisplayString()) // 'stella@satelliteA (abc123...)'\n\n// Verify signature\nconst isValid = await meshId.verify('message', signature)\n```\n\n### Auth Helpers\n\n```typescript\nimport { createAuthMessage, verifyAuthMessage } from './index.js'\n\n// Create signed auth message\nconst authMsg = await createAuthMessage(meshPass, 'stella@satelliteA')\n// Returns: { meshId, nonce, timestamp, signature }\n\n// Verify auth message\nconst isValid = await verifyAuthMessage(authMsg, publicKeyHex)\n```\n\n## CLI Commands\n\n```bash\n# Generate new MeshPass\nnpx tsx meshpass.ts generate\n\n# Show MeshPass info\nnpx tsx meshpass.ts info\n\n# Export for backup/transfer\nnpx tsx meshpass.ts export > backup.json\n\n# Import from file\nnpx tsx meshpass.ts import backup.json\n\n# Sign a message\nnpx tsx meshpass.ts sign \"Hello, mesh!\"\n\n# Verify signature\nnpx tsx meshpass.ts verify \"Hello, mesh!\" \"signature-hex\" \"pubkey-hex\"\n\n# Generate auth message\nnpx tsx meshpass.ts auth stella@satelliteA\n```\n\n## Security Notes\n\n- **Private keys never leave your machine** - only public keys are shared\n- **Messages are signed, not encrypted** - signatures prove authenticity, not privacy\n- **Passphrase protects your MeshPass file** - choose a strong passphrase\n- **Signatures have timestamps** - prevents replay attacks (5 minute window)\n- **Rate limiting** - The Gate protects against spam and abuse\n\n## Migration from Tailscale\n\nThe MeshPass system is designed for backward compatibility:\n\n1. **Tailscale agents keep working** - no changes needed\n2. **Gradual migration** - agents can choose Tailscale OR The Gate\n3. **Signature optional** - messages work with or without signatures\n4. **Dual transport** - agents can connect via both Tailscale and The Gate\n\n## File Locations\n\n- **MeshPass**: `~/.manifold/meshpass.json`\n- **Configuration**: Environment variables or config files\n- **Logs**: Console output (use `debug: true` in configs)\n\n## Environment Variables\n\n- `MESHPASS_PASSPHRASE` - Passphrase for CLI operations\n- `MESHPASS_PATH` - Custom path to MeshPass file\n- `GATE_DEBUG` - Enable debug logging for The Gate\n\n## Examples\n\nSee `federation/examples/` for complete examples:\n- Agent connecting via The Gate\n- Setting up a public gateway\n- Migration from Tailscale federation\n"
