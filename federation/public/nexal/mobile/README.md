# Nexal Mobile

Mobile implementations and optimizations for the Nexal federation interface.

## Structure

- `app/` - Native mobile app components (React Native, Flutter, etc.)
- `pwa/` - Progressive Web App implementation
- `responsive/` - Mobile-optimized responsive web versions
- `../shared/` - Shared components and utilities across platforms
- `../desktop/` - Desktop/web versions and development files

## Mobile Features

The mobile version focuses on:
- Touch-optimized controls for 3D navigation
- Simplified UI for smaller screens
- Optimized performance for mobile devices
- Offline capabilities (PWA)
- Native mobile app integration

## Development

Each subdirectory contains platform-specific implementations while sharing core federation visualization and networking components from the shared directory.
