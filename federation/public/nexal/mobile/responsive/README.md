# Mobile Responsive Nexal

Mobile-optimized responsive web version of the Nexal federation interface.

## Features

- Touch-friendly 3D navigation controls
- Responsive layout for phones and tablets
- Simplified UI panels that work well on small screens
- Optimized WebGL performance for mobile GPUs
- Gesture-based interaction with the 3D visualization

## Files

- `index.html` - Main mobile-responsive interface
- `mobile.css` - Mobile-specific styles and layout
- `touch-controls.js` - Touch interaction handlers
- `mobile-3d.js` - Mobile-optimized 3D scene setup

## Performance Considerations

- Reduced particle counts for mobile performance
- Simplified shaders for mobile GPUs
- Touch-optimized camera controls
- Efficient memory management for mobile devices
