# Nexal Native Mobile App

Native mobile application implementations for iOS and Android.

## Platforms

- React Native (cross-platform)
- Flutter (cross-platform)
- Native iOS (Swift)
- Native Android (Kotlin)

## Features

- Native 3D rendering with Metal/Vulkan
- Push notifications for federation events
- Background federation mesh monitoring
- Native device integration (haptics, sensors)
- Optimized touch controls for 3D navigation

## Structure

```
react-native/    # React Native implementation
flutter/         # Flutter implementation  
ios/            # Native iOS app
android/        # Native Android app
shared/         # Shared native components
```

## Development

Each platform directory contains the complete mobile app project with platform-specific build configurations and native integrations.

## Federation Integration

Native apps connect to the federation mesh using optimized mobile protocols and maintain persistent connections for real-time updates.
