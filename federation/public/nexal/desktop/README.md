# Nexal Desktop Versions

Desktop/web implementations and development files for the Nexal federation interface.

## Current Files

- `index-*.html` - Various development versions and iterations
- `fog-windows-v2.html` - Windows-style GUI with fog animation
- `clean-fog-windows.html` - Clean implementation
- `working-fog.html` - Stable fog animation version
- Development and backup versions

## Features

- Full desktop 3D visualization with WebGL
- Complex mathematical topologies (Klein bottles, Möbius strips)
- Windows-style GUI overlay
- Advanced particle systems and shaders
- Mouse and keyboard navigation
- High-performance rendering for desktop GPUs

## Main Interface

The primary desktop interface is served from the parent directory (`../index.html`) for production use, while this directory contains development versions and iterations.

## Development Workflow

Development versions are tested here before being promoted to the main production interface. This allows for safe experimentation with new features and UI improvements.
