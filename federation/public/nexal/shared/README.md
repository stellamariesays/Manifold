# Shared Nexal Components

Shared components, utilities, and core functionality used across desktop and mobile platforms.

## Components

- `federation-client.js` - Core federation mesh client
- `3d-core.js` - Base 3D visualization engine
- `math-utils.js` - Mathematical topology utilities (Klein bottles, Möbius strips)
- `api-client.js` - Federation API communication
- `constants.js` - Shared configuration and constants

## 3D Visualization Core

- Three.js scene management
- Klein bottle and Möbius strip geometries
- Spider web network visualization
- Orbital dynamics system
- Particle effects and shaders

## Federation Integration

- Mesh connectivity protocols
- Agent data synchronization
- Real-time capability routing
- Cross-platform event handling

## Usage

These components are imported and adapted by both desktop and mobile implementations, providing consistent core functionality while allowing platform-specific optimizations.
