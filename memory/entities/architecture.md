
## Power Levels — Group Chat Behavior (Hal's terminology)

These describe how Stella behaves in **group chats**. Cumulative layers.

- **Goku** — default chatbot mode. Few LLM calls. No Manifold, no Numinous. KISS.
- **SSJ1** — Goku + pull Manifold context into responses (mesh, atlas, Sophia scores).
- **SSJ2** — SSJ1 + Numinous (dark circles as live Void processes).
